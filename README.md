# Toolsyou

Toolsyou est un site regroupant de nombreux outils gratuits accessibles directement depuis un navigateur.

## 🌐 Site

https://toolsyou.fr

## ✨ Fonctionnalités

- Outils PDF
- Outils images
- Convertisseurs
- Outils pratiques du quotidien
- Et bien d'autres fonctionnalités à venir

## 🚀 Fonctionnement

La majorité des outils fonctionnent directement dans le navigateur de l'utilisateur, sans envoyer les fichiers vers un serveur.

## 🤝 Contribution

Le projet est open source.  
Vous pouvez proposer des améliorations, corriger des bugs ou ajouter de nouvelles fonctionnalités via des Pull Requests.

## 👨‍💻 À propos du développement

Ce projet a été développé principalement depuis un téléphone, avec l'aide de l'intelligence artificielle pour accélérer certaines étapes de développement.

J'utilise notamment des outils comme **TrebEdit**, **Termux** et d'autres outils mobiles pour coder et tester le projet.

Je suis encore en apprentissage dans le développement web, donc s'il y a des erreurs, des améliorations possibles ou des pratiques que je pourrais mieux appliquer, n'hésitez pas à me les signaler. Les conseils et les retours sont les bienvenus, l'objectif est d'apprendre et de rendre ce projet meilleur avec l'aide de la communauté.
