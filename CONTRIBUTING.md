# 🤝 Contribuer à Toolsyou

Merci de vouloir contribuer au projet !

## 💡 Proposer une idée

Si vous avez une idée d'amélioration, une nouvelle fonctionnalité ou un outil à ajouter, vous pouvez ouvrir une **Issue** sur GitHub.

## 🛠️ Modifier le projet

1. Faites un **Fork** du projet.
2. Créez une nouvelle branche.
3. Effectuez vos modifications.
4. Envoyez une **Pull Request** avec une explication des changements.

## 🐛 Signaler un problème

Si vous trouvez un bug ou une erreur, n'hésitez pas à le signaler. Les retours sont les bienvenus et permettent d'améliorer le projet.

## 📚 Apprendre ensemble

Je suis encore en apprentissage du développement web, donc tous les conseils, corrections et suggestions d'amélioration sont appréciés.
